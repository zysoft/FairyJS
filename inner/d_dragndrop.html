Reference Guide :: fjs_dragndrop.js

Basic information

Drag&Drop plugin is accessible via $$.fjs.dragndrop and enables dragging and dropping items for your website. Of course it won't let you handle this without JavaScript knowledge, but I hope it will simplify your work.

HTML attributes provided

Attribute Applicable to Value Description
data-fjs-draggable Any Dragging containment Makes element draggable and sets its dragging containment
See jQuery.ui docs
data-fjs-draggable-cursor data-fjs-draggable CSS value for "cursor" property Sets cursor for object while dragging
data-fjs-draggable-revert data-fjs-draggable valid
invalid
true (means "always")		 Sets item to restore its position after it is dragged to droppable container.
See jQuery.ui docs
data-fjs-draggable-clone data-fjs-draggable Any Sets options to clone draggable item on drag and then drag the clone.
data-fjs-drop-to data-fjs-draggable Container ID Sets the container to drop element to. Should match data-fjs-droppable
data-fjs-droppable Any Container ID Makes element droppable and sets the container ID to accept drops
Used in conjunction with data-fjs-drop-to
data-fjs-highlight-class data-fjs-droppable CSS class name Sets the CSS class to highlight container while dragging

Configuration options

None
