Reference Guide :: fjs_core.js

Basic information

Core object is named $$.fjs. FairyJS depends on jQuery and provides a shortcut for jQuery object named $$. This helps keeping shortcut short in case when jQuery is used in non conflicting mode.

Core object is nothing but container for plugins which build the core as of your needs. Every plugin can be added or removed without breaking up things. Except the plugins that are dependant on other plugins.

None
