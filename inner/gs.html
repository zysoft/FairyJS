Getting started with FairyJS

You don't have to be JavaScript specialist to start using FairyJS. It's fairly simple!

Including FairyJS

First you need to get jQuery and either All-In-One pack or core file and the set of plugins you need for your site.

All project code is available from here - FairyJS on Github

Available files are:

File Description Comments
fjs-min.js FairyJS All-In-One package Default
fjs_core.js FairyJS core Required
fjs_lang.js Localization support plugin  
fjs_forms.js Forms handling plugin  
fjs_validate.js Forms validation plugin Requires fjs_forms.js
fjs_dragndrop.js Drag&Drop support plugin Requires jQuery.ui
fjs_facebook.js Facebook integration plugin

Configuring FairyJS and plugins

FairyJS can be easily configured. By default it has only two options - debugMode and verboseMode. See fjs_core.js docs for detailed explanation.

Other options are provided by plugins like fjs_lang.js.
