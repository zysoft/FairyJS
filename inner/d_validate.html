Reference Guide :: fjs_validate.js

Basic information

Validate plugin is accessible as $$.fjs.validate. It provides HTML form fields validation mechanisms and contains some built-in validations. Depends on Forms plugin and won't start without.

HTML attributes provided

Attribute Applicable to Value Description
data-fjs-validate HTML form inputs blur | keypress | click Turns on validation and specifies when to validate field contents. Empty value means on submit.
data-fjs-required HTML form inputs Any Indicates that field is mandatory
data-fjs-validator HTML form inputs Comma separated list of validators List of validators to apply
data-fjs-validator_param HTML form inputs Depends on parameter meaning Validator parameter
data-fjs-validator_error_message HTML form inputs String Validator custom error message
