How does it work



    When you specify language in your HTML tag by adding data-fjs-lang, you set it as the default language.
    At this stage your language file is loaded (if you didn't load it by yourself).
    Then each data-fjs-localizable entity is translated.




    All translations are made from a "basic" language which you can choose by your own.
    This means all language files should be built accordingly to your choice. If you choose
    English as a basic language - you should perform all translations from English.




    For further overview of features, please see fjs_lang.js documentation.
