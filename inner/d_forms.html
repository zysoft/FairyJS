Reference Guide :: fjs_forms.js

Basic information

Forms plugin is accessible as $$.fjs.forms. It provides some basic functionality for forms handliding and can be used with Validation plugin to allow client-side forms validation.

HTML attributes provided

Attribute Applicable to Value Description
data-fjs-form-ajax HTML <form> Any Tells to use Ajax for form submission

Configuration options

None

Generated events

jQuery Object

Event
jQuery Object

Form which this event is triggered for
Object | String
response
Server response. It can be string or object if server returned headers for the jQuery understandable format (like JSON or XML)
Gets fired when form ajax submit succeeds (server returned code 200) 
                        $$.fjs.subscribe('org.fjs.form.submit.success',
                                        function($ev, $form, response){
                                            alert('Submission ok. Server response:'+response);
                                        }
                        );
